A modern social media platform for discovering and sharing viral content from TikTok, Instagram Reels, YouTube Shorts, and more.

## Features

- **Multi-Platform Support**: Share videos from TikTok, Instagram, YouTube, Facebook
- **Infinite Scroll**: TikTok-like feed with autoplay videos
- **Social Interactions**: Like, comment, and share posts
- **User Profiles**: Personal profiles with post history and liked content
- **Mobile-First**: Optimized for mobile devices
- **Real-time Updates**: Live like and comment counts

## Tech Stack

- **Frontend**: Next.js 14, React, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage)
- **Deployment**: Vercel (recommended)
